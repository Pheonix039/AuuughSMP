Welcome to my server homepage! Auuugh SMP is a Minecraft: Java Edition server, available on versions 1.21.10 and onward.
The server consists of 5 dimensions: The Overworld, The Nether, The End, The Void (Skyblock Dimension), and The Deeper Dark.
The Server has a player count of 60, but can support up to 100 if needed.
The community for my server is small, but tightknit, with tons of players from all walks of life!
If playing Minecraft with me and my community sounds intriguing to you, don't hesitate to join! We love seeing new members join!
